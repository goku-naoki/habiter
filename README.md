# Habiter

Habiter(ハビター)はrailsとvue.jsで作られた習慣管理アプリです。
**"習慣形成を行って、最高な毎日を"**と言うコンセプトの下、
英語「習慣」を意味する"habit",「生きる、住む」を意味するフランス語 "habiter"を
掛け合わせて命名しました。
目標があっても行動に移せないと言う問題を解決する為に、小さな一歩からサポートします。

##  機能一覧

1. user(ユーザー),habit(習慣),habit_user(ユーザーに紐づく習慣)のCRUD
2. ログイン、ログアウト、ゲスト用自動ログイン
3. 習慣達成チェック機能
4. 画像アップロード機能
5. ユーザーフォロー機能
6. ユーザー検索
